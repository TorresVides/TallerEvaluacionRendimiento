# TallerEvaluacionRendimiento
Con ayuda del Makefile se compila, luego de esto ya podemos obtener los datos necesarios para graficar que tendran como extencón/tipo de archivo .csv

El código presentado en el repositorio está ordenado de la siguiente manera:

src/ (todos los .c)
include/ (todos los .h)
data/ (.csv)
scripts/ (.pl)
